Input text. Make sure everything's the way you want it, because you won't be able to make any changes to the initial wording after you hit "Save".

Once the text is saved, you will be presented with a set of controls that will allow you to pick out nouns, verbs, adverbs and adjectives. When you've selected the ones you need, click "Submit" to send the data.

The server will randomly replace words with one another in each group and present you with the text resulting therefrom.

For outputting poetry, you need to pick a number upon which the length of lines in your poem is going to depend. The algorithm is loosely based on the prosody of ancient Greeks and Romans. Poetry mode is also going to remove all punctuation from the input text.

Non-English characters, numbers and some offending punctuation will be removed for both modes. Capitalisation is not retained either.

For more interesting results, combine this tool with the Random Paragraph Creator.

Pick one of the parts of speech above and start clicking away at the words to the left. Selecting a word adds it to the corresponding list, selecting it again – removes it.

Push "Submit" down below once you're finished.
